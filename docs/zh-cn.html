强大的文本转语音功能

专业音频内容创作和无障碍访问所需的一切功能

Azure 语音服务

访问60多种语言的200多个高质量语音，支持先进的神经语音合成技术和企业级可靠性。

语音定制

选择说话风格、调整语速、音调和音量。为增强语音输出和创意内容选择角色扮演字符。

VS Code 集成

通过简单的键盘快捷键和右键菜单集成，无缝转换选定文本或整个文档。

多语言支持

英文和中文界面完全本地化，具有可扩展的国际化架构和自动语言检测。

实时处理

实时进度反馈和大文档处理的智能分块功能，优化性能和内存管理。

智能音频管理

自动文件命名、格式选择（MP3、WAV）和有序的音频输出处理，智能分块管理。

快速安装

在不到2分钟内准备好您的文本转语音扩展

1

从 VS Code 市场安装

在 VS Code 扩展中搜索 "Speechify" (Ctrl+Shift+X) 或点击上方安装按钮。

2

配置 Azure 语音服务

在 VS Code 设置中设置您的 Azure 语音服务订阅密钥和区域以获得完整功能。

3

开始转换文本

选择文本并使用 Ctrl+Shift+P → "Speechify: Convert to Speech" 开始创建音频内容。
