快速安装

在不到2分钟内准备好您的文本转语音扩展

1

从 VS Code 市场安装

在 VS Code 扩展中搜索 "Speechify" (Ctrl+Shift+X) 或点击上方安装按钮。

2

配置 Azure 语音服务

在 VS Code 设置中设置您的 Azure 语音服务订阅密钥和区域以获得完整功能。

3

开始转换文本

选择文本并使用 Ctrl+Shift+P → "Speechify: Convert to Speech" 开始创建音频内容。
